# My Web Project 🌐

Простое и функциональное веб-приложение на фреймворке Django, состоящее из главной страницы и раздела с контактами.

## 📋 Особенности
* **Главная страница**: Приветственная часть с описанием проекта.
* **Страница Контакты**: Форма обратной связи или контактные данные.


## 🛠 Технологии
* **Python 3.x**
* **Django 4.2+**
* **HTML5 / CSS3 / JS /Bootstrap**
